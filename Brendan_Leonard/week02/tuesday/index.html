Zach

Zach's surname is rendered in a font called 'Toyota iQ font'. The font was created by a team involving Zach, 2 typographers and a stunt driver. The stunt driver drove separate patterns for all letters and characters using a tiny Toyota iQ car. Zach designed software that took a real-time feed from a movie camera suspended from a crane, the software converting the cars tracks into the various elements of the font.
